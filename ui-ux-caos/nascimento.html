🎓 O que você está praticando nessa página (e sofrendo)
  • Anti-padrões de UX — cada step viola uma regra básica de usabilidade
  • Event Listeners — click, mouseover, input, mouseup, keydown para criar o caos
  • Manipulação do DOM — querySelector, classList, textContent, style, innerHTML
  • IndexedDB — persistência assíncrona com `db.js` e `controller.js`
  • Lógica JS Avançada — contadores, aleatoriedade, posicionamento dinâmico, timers
  • Resiliência e Paciência — habilidades essenciais para qualquer desenvolvedor!
